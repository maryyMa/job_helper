## 功能说明

根据用户的个人经历和目标岗位JD，生成一份全面的面试准备指南。

## 输入

- 用户个人经历（工作经历、项目经验、技能等）
- 目标岗位JD（用于定制化问题预测）

## 输出

### 模块一：高频面试问题预测（10个）

每个问题包含：
- **问题**：面试官可能问的问题
- **考察意图**：这个问题想考察什么能力
- **STAR回答**：基于用户经历，给出具体的回答示例
- **避坑指南**：回答时要避免什么

问题类型覆盖：
1. 自我介绍（1分钟版）
2. 行为面试题（3个）：团队协作、冲突处理、压力应对
3. 岗位专业题（3个）：针对岗位核心能力
4. 职业规划题（1个）
5. 薪资期望题（1个）
6. 反问环节（1个）

### 模块二：弱点应对策略

针对用户经历与岗位要求的差距，提供：
1. 面试官可能追问的方向
2. 巧妙应对的回答模板
3. 如何将话题引导到优势领域

### 模块三：薪资谈判策略

1. 该岗位的市场薪资区间（需使用websearch查询）
2. 基于用户经历的定价建议
3. 被问期望薪资时的回答话术
4. 如何委婉争取更高薪资

### 模块四：面试前Checklist

- 需要了解的公司信息（业务、产品、文化、近期动态）
- 需要准备的材料
- 需要复习的知识点

---

## 执行规范

### 问题预测要求
1. 必须基于岗位JD预测问题，不能泛泛而谈
2. STAR回答必须基于用户真实经历，禁止编造
3. 避坑指南要具体可操作

### 薪资查询要求
1. 必须使用websearch查询真实薪资数据
2. 搜索来源：招聘平台薪资信息、行业薪资报告
3. 禁止编造薪资范围

### 公司调研要求
1. 必须使用webfetch访问公司官网
2. 必须使用websearch搜索公司评价、新闻
3. 禁止编造公司信息

### 质量检查
- [ ] 问题预测是否覆盖高频考点？
- [ ] STAR回答是否基于真实经历？
- [ ] 弱点应对是否具有可操作性？
- [ ] 薪资策略是否基于真实数据？
- [ ] 公司信息是否准确？
