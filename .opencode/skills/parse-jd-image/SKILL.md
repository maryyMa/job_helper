## 功能说明

当用户提供的JD是截图时，使用此skill识别图片内容，提取结构化的岗位信息。

## 使用场景

- 用户发送JD截图
- 用户发送包含岗位信息的图片

## 工作流程

### 第一步：识别图片

使用支持图片的模型（mimo-v2.5）识别图片内容。

**需要识别的信息：**
- 公司名称
- 岗位名称
- 岗位描述
- 任职要求
- 薪资范围（如有）
- 工作地点（如有）

### 第二步：自动检查

识别完成后，自动检查以下内容：

**完整性检查**
- [ ] 是否识别出公司名称？
- [ ] 是否识别出岗位名称？
- [ ] 是否识别出岗位描述？
- [ ] 是否识别出任职要求？

**准确性检查**
- [ ] 识别的文字是否有明显错误？
- [ ] 关键信息（公司名、岗位名）是否合理？
- [ ] 是否有乱码或识别失败的内容？

**检查结果处理**
- 如果检查通过：直接输出结构化结果，继续后续流程
- 如果检查不通过：尝试修正或告知用户哪些信息识别不完整

### 第三步：结构化输出

将识别的信息整理成标准格式：

```
## 识别结果

### 公司信息
- 公司名称：xxx
- 公司规模：xxx（如有）
- 所属行业：xxx（如有）

### 岗位信息
- 岗位名称：xxx
- 工作地点：xxx
- 薪资范围：xxx

### 岗位描述
[粘贴识别出的岗位描述]

### 任职要求
[粘贴识别出的任职要求]

### 加分项
[粘贴识别出的加分项，如有]
```

---

## 注意事项

1. **必须使用支持图片的模型**：调用此skill时，需要切换到mimo-v2.5
2. **自动检查**：识别后自动检查完整性和准确性
3. **识别不完整时**：告知用户哪些信息未能识别，请用户补充
4. **多张图片**：如果用户发送多张图片，逐一识别并合并信息
